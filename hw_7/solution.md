# Разложение x⁷ - 1 на множители над F₂

В поле F₂ выполняется x⁷ - 1 = x⁷ + 1. Разложение на неприводимые множители:

\[
x^7 + 1 = (x + 1)(x^3 + x + 1)(x^3 + x^2 + 1)
\]

# Построение 6 порождающих полиномов

1. \( g_1(x) = x + 1 \)  
2. \( g_2(x) = x^3 + x + 1 \)  
3. \( g_3(x) = x^3 + x^2 + 1 \)  
4. \( g_4(x) = (x+1)(x^3+x+1) = x^4 + x^3 + x^2 + 1 \)  
5. \( g_5(x) = (x+1)(x^3+x^2+1) = x^4 + x^2 + x + 1 \)  
6. \( g_6(x) = (x^3+x+1)(x^3+x^2+1) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1 \)

## Характеристики полученных кодов

Для каждого порождающего полинома \(g(x)\) степени \(r\) циклический код длины \(n=7\) имеет размерность \(k = n - r\). Минимальное расстояние \(d\) определяется известными кодами.

| Полином | Степень \(r\) | Размерность \(k\) | Минимальное расстояние \(d\) | Тип кода |
|---------|---------------|-------------------|------------------------------|-----------|
| \(g_1(x)=x+1\) | 1 | 6 | 2 | Код с проверкой на чётность |
| \(g_2(x)=x^3+x+1\) | 3 | 4 | 3 | Код Хэмминга (7,4) |
| \(g_3(x)=x^3+x^2+1\) | 3 | 4 | 3 | Код Хэмминга (7,4) |
| \(g_4(x)=x^4+x^3+x^2+1\) | 4 | 3 | 4 | Дуальный коду Хэмминга, симплексный код (7,3) |
| \(g_5(x)=x^4+x^2+x+1\) | 4 | 3 | 4 | То же, эквивалентный (7,3) код |
| \(g_6(x)=\sum_{i=0}^{6}x^i\) | 6 | 1 | 7 | Код с повторением |

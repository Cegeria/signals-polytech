# Задание 2 

Двоичный код Голея $(23,12)$ — это линейный код длины $n=23$, размерности $k=12$ над полем $\mathbb{F}_2$. 

## Минимальное расстояние кода

Минимальное расстояние $d$ кода Голея $(23,12)$ равно **7**. Это означает, что код может обнаруживать до 6 ошибок и исправлять любые $t = \left\lfloor\frac{d-1}{2}\right\rfloor = 3$ ошибки. Значение $d=7$ хорошо известно из теории кодирования и может быть подтверждено, например, через проверочную матрицу или свойства весов кодовых слов (код является циклическим, и его веса кратны 4, причём минимальный ненулевой вес равен 7).

## Проверка выполнения равенства для кода Голея (23,12)

Для кода $(23,12)$ имеем:
- $n = 23$,
- $k = 12$,
- $t = 3$ (поскольку $d=7$).

Вычислим объём шара радиуса $3$:

$$ 
V = \sum_{i=0}^{3} \binom{23}{i} = \binom{23}{0} + \binom{23}{1} + \binom{23}{2} + \binom{23}{3}.
$$

Значения биномиальных коэффициентов:
- $\binom{23}{0} = 1$,
- $\binom{23}{1} = 23$,
- $\binom{23}{2} = \frac{23\cdot22}{2} = 253$,
- $\binom{23}{3} = \frac{23\cdot22\cdot21}{6} = 1771$.

Сумма:
$$ 1 + 23 = 24, \quad 24 + 253 = 277, \quad 277 + 1771 = 2048. $$

Таким образом, $V = 2048 = 2^{11}$.

Теперь вычислим правую часть границы:
$$ 2^{n-k} = 2^{23-12} = 2^{11} = 2048. $$

Обе части совпадают:
$$ \sum_{i=0}^{3} \binom{23}{i} = 2^{23-12}. $$

Следовательно, для кода Голея $(23,12)$ граница Хемминга достигается с равенством.

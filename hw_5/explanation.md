# Описание алгорита
1. Для заданного \(r\) строим все столбцы \(H\) (все ненулевые двоичные векторы длины \(r\)).
2. Перебираем все ненулевые \(\mathbf{a}\) (от \(1\) до \(2^r-1\)).
3. Для каждого \(\mathbf{a}\) вычисляем кодовое слово \(\mathbf{c} = \mathbf{a} H\) 
4. Подсчитываем вес Хэмминга полученного слова.
5. Сравниваем вес с ожидаемым значением \(2^{r-1}\). Если все веса совпадают, то дуальный код является симплексным.

Программа успешно выполняет эту проверку для \(r = 2,3,4,5\), подтверждая требуемое свойство.
